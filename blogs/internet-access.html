科学上网完整指南

本文汇总科学上网的基本概念、环境配置步骤与常见工具，帮助你快速完成从 0 到 1 的搭建。 配置材料与订阅可以在工具交流群中获取，适合 Windows / macOS / Android / iOS 等平台。

⚠️ 合规与前置说明

  • 请遵守当地法律法规，仅用于学习、科研和合规访问。
  • 谨慎保管个人账号与订阅链接，不要公开分享。
  • 如访问速度不理想，可尝试更换节点、协议或切换网络（WIFI/蜂窝）。

📚 基本术语

  • 订阅（Subscription）：包含多个节点配置的链接或文件，导入客户端即可使用。
  • 节点：实际的代理服务器，国家/线路不同速度也不同。
  • 系统代理：让全局应用流量都经过客户端（或设置分应用/分域名规则）。

🖥️ Windows / macOS

  • 推荐客户端：Clash Verge Rev、SagerNet 桌面版、Sing-Box GUI。
  • 步骤：
    1. 下载安装客户端。
    2. 在“订阅/配置”处粘贴你的订阅链接或导入订阅文件。
    3. 更新订阅，选择延迟更低的节点（可测速）。
    4. 开启系统代理（或规则模式），浏览器与应用即可走代理。
  • 常见问题：证书安装失败、端口占用（更换本地端口）、公司/校园网络限制（尝试不同协议或开启混淆）。

📱 Android

  • 推荐客户端：Surfboard、Clash Meta for Android、SagerNet。
  • 步骤：安装 → 导入订阅 → 选择节点 → 开启“代理开关”。
  • 技巧：
    • 分应用代理：仅让指定 App 走代理，降低耗电与流量。
    • 分流规则：常见国内域名直连，海外域名走代理。

🍎 iOS / iPadOS

  • 常用客户端：Shadowrocket、Loon、Quantumult X（需使用支持的地区账号）。
  • 步骤：App Store 安装 → 粘贴订阅 → 更新 → 选节点 → 打开“连接”。
  • 提示：如账号地区限制，可在交流群了解合规获取方式与注意事项。

🚀 测速与优化

  • 客户端内置测速：延迟（ms）、下载/上传速度。
  • 选择同运营商/同地区线路通常更稳。
  • 网络不稳时：切换节点协议（如 Hysteria / Reality / VLESS / Trojan 等）、更换 DNS、或重启路由器。

🧰 工具清单（可在交流群获取）

  • 移动：Surfboard / Clash Meta for Android / Shadowrocket
  • 辅助：测速站（speedtest / fast）、DNS 工具（NextDNS / AdGuard）
