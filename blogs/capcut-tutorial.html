剪映使用教程（手机端 & PC 端）

本文提供剪映（CapCut）在手机端与 PC 端的高效使用教程，涵盖常见功能与实用技巧，帮助你快速上手高效创作。

一、手机端（iOS / Android）

  • 获取方式：App Store / 各大安卓应用商店搜索“剪映”。
  • 新手上手：
    1. 新建项目 → 选择画布比例（9:16/1:1/16:9）。
    2. 导入素材（相册/相机） → 按时间线排列。
    3. 基础编辑：分割（剪刀）、删除、速度（变速/曲线）、音量、倒放。
    4. 转场与字幕：在片段间添加转场，打开“文字”添加字幕与描边/阴影。
    5. 音频：音乐库、提取配乐、消除人声/提取人声（部分功能需登录或会员）。
    6. 画面优化：滤镜、调色、锐化、降噪、防抖。
  • 导出设置：分辨率（1080p/4K）、码率（8–20Mbps）、帧率（30/60fps），按平台要求选择。
  • 提升效率：预设模板、收藏常用字体/贴纸，启用“自动字幕”，素材命名清晰。

二、PC 端（Windows / macOS）

  • 获取方式：访问剪映官网/应用商店，下载桌面版。
  • 典型工作流：
    1. 新建项目 → 设定分辨率与帧率 → 导入素材（视频/图片/音频）。
    2. 时间线剪辑：J/L 快捷键预览，S 分割，Delete 删除，Ctrl/Cmd+Z 撤销。
    3. 多轨道：旁白、BGM、音效独立轨；画中画用于贴纸/叠加。
    4. 字幕与特效：自动字幕校对、关键帧做动效、蒙版与混合模式。
    5. 调色与降噪：使用 LUT，音频降噪/均衡器，保证素材清晰。
  • 发布规范：导出前检查音画同步、响度峰值（-14 LUFS 左右）、标题/封面尺寸。
会员与素材库：如需高阶素材/特效/商用授权，请使用官方会员或合法素材平台（Pixabay、Pexels、Icons8、Envato 等）。

三、常见问题

  • 导出卡住/失败：检查磁盘空间、更新显卡驱动、降低码率或关闭硬件加速再试。
  • 字体不一致：收集字体打包同步；避免在他机缺字导致样式变化。
  • 移动端到桌面协作：在手机端导出“项目包/素材包”，桌面端导入继续编辑。
