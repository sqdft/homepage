B站视频下载网页程序

基于 Node.js 的简易 B站视频下载网页程序，方便用户快速下载 B站视频。

环境需求

  • 已安装 Node.js
  • 通过命令 node -v 和 npm -v 查看版本

安装步骤

  1. 初始化项目：
    npm init -y
  2. 安装依赖包：
    npm install express axios cors
  3. 将项目文件解压到工作目录

项目文件下载

项目压缩包下载链接（百度网盘）：

bili-grabber.zip（提取码：kgca）

启动服务

node server.js

启动成功后，打开浏览器访问 http://localhost:端口号 使用视频下载功能。

说明：该程序基于 Express 框架，使用 Axios 请求 B站接口，支持 CORS 跨域访问。适合本地或服务器部署。
